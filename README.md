## Installation

### Prerequisites
- Node.js 16+
- npm or yarn

### Development Setup
1. Clone the repository
```bash
git clone https://github.com/baturkacamak/media-master-ultimate-plus
cd media-master
```

2. Install dependencies
```bash
npm install
```

3. Start the application
```bash
npm run dev
```
